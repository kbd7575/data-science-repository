## Projects

### 1. Responder 6 Time Series Prediction
- Built predictive models on a large financial dataset (42M+ rows)
- Focused on forecasting a proprietary target variable
- Addressed feature selection and data leakage
- Models: XGBoost, Random Forest
- Result: Achieved performance close to top competition score

### 2. Diabetes Prediction
- Developed a classification model to predict diabetes diagnosis
- Evaluated using accuracy, precision, recall, and F1 score
- Models: Logistic Regression, Random Forest, Gradient Boosting
- Result: Achieved 92 percent accuracy and strong F1 performance

### 3. Manufacturing Defect Detection
- Built classification models to identify defective materials
- Compared baseline vs feature engineered models
- Models: Random Forest, LightGBM
- Result: Achieved high ROC AUC and improved recall performance

### 4. Consumer Complaint Text Classification
- Applied text mining techniques to classify complaint narratives
- Compared rule-based, SVM, BERT, and LLM approaches
- Evaluated using accuracy and macro F1 score
- Result: BERT and SVM achieved highest performance
